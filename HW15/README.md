# HW15 - Servo Design / Haptic Paddle First Draft

This folder contains my first CAD draft for the haptic paddle mechanical assembly.

## Assignment summary
The HW15 design includes the main mechanical components for a haptic paddle:
- RC servo and servo horn
- paddle arm
- load cell mounting area
- thimble/finger clamp
- AS5600 magnetic encoder and magnet location

## Files
- `assembly_overview.png` - labeled assembly overview of the first draft.
- `first_version_parts_render.png` - CAD-style layout of the first printed parts.
- `design_notes.md` - short design explanation and lessons expected from the first print.
- `haptic_paddle_first_draft.scad` - OpenSCAD source file for the simple first-draft model.
- `paddle_arm.scad` - printable paddle arm concept.
- `load_cell_adapter.scad` - adapter plate for load cell mounting.
- `thimble_clamp.scad` - finger/thimble clamp concept.
- `as5600_mount.scad` - AS5600 magnetic encoder bracket concept.
- `servo_base_plate.scad` - servo base plate concept.

## Notes
This is a first draft. The first print is expected to need small changes after testing fit, backlash, screw length, and clearances.
